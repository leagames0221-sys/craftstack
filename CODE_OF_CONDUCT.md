# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be welcoming and inclusive.
- Critique code, not people.
- Assume good faith; ask before assuming malice.
- No harassment, discrimination, or personal attacks.

## Enforcement

Report concerns privately via the security contact in [SECURITY.md](SECURITY.md). I take reports seriously and respond within 72 hours.

## Scope

This code applies in issues, pull requests, discussions, and any other space associated with the project.
